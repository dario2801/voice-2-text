# Voice-2-Text

Audio transcription and translation tool powered by Whisper AI. Upload any audio file and get an instant English transcription with automatic language detection.

## Features

- Automatic language detection
- Audio-to-English transcription and translation
- Supports OGG, MP3, WAV, M4A, FLAC, WebM, and AAC formats
- Up to 25 MB file size
- Minimal dark UI built with Next.js

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **AI**: OpenAI Whisper
- **Fonts**: DM Mono + Instrument Serif

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
